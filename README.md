# Accredian Enterprise Platform

> 🚧 **Status: Ongoing / Work in Progress** – *This project is currently under active development. Features and UI are being continuously refined.*
![Accredian UI Demonstration](./frontend/public/demo.webp)

**Accredian Enterprise Platform** is a cutting-edge corporate training and development landing page built to empower modern workforces with advanced learning solutions in Data Science, AI, and Product Management. The platform is designed to convert leads, explain our unique 3-step value proposition, and establish corporate credibility.

## 🚀 Key Features

- **Next-Gen Messaging & Branding:** Professional, enterprise-focused copy and design to engage global organizations.
- **The Accredian Edge:** Dedicated feature highlights for Tailored Solutions, Innovative Frameworks, Diverse Offerings, and Flexible Delivery.
- **3-Step Process Automation:** A clean, easy-to-understand breakdown of our workflow (Skill Gap Analysis, Custom Plan, Flexible Delivery).
- **Corporate Partner Showcase:** High-fidelity presentation of key enterprise partners (Reliance, HCL, IBM, etc.).
- **Dynamic Testimonials:** Seamless integration of authentic enterprise feedback via Next.js serverless API routes.
- **High-Performance Architecture:** Built from the ground up for extreme speed using Next.js (Turbopack) and custom image optimization strategies.

## 💻 Tech Stack

- **Frontend Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS (with custom utility classes and animations)
- **Bundler:** Turbopack for ultra-fast local development
- **Deployment:** Vercel (Production-Ready)
- **Icons & Media:** Inline SVGs and Next.js highly-optimized Images

## 🛠️ Local Development Setup

To run this project locally, ensure you have Node.js (v20+) installed.

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd "Accredian Enterprise/frontend"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   *Note: Next.js + Turbopack can be memory intensive. We recommend running with increased Node memory if you encounter OOM issues.*
   ```powershell
   $env:NODE_ENV="development"; $env:NODE_OPTIONS="--max-old-space-size=4096"; npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
